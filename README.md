# Neon Bubble Shooter

A browser-based bubble shooter game with a neon aesthetic.

## How to Play

- **Aim**: Click/tap and drag to aim
- **Shoot**: Release to fire the bubble
- **Match**: Connect 3+ bubbles of the same color to pop them
- **Win**: Clear all bubbles from the board
- **Lose**: Let bubbles reach the bottom

## Features

- Hexagonal grid layout
- Wall bouncing physics
- Floating bubble detection (bonus points for dropping disconnected bubbles)
- Particle effects
- Sound effects
- Touch and mouse support
- Score tracking

## Running

Open `index.html` in a browser, or serve locally:

```bash
npx serve .
```

## Tech

- Vanilla JavaScript with HTML5 Canvas
- Web Audio API for sound effects
- No external dependencies
